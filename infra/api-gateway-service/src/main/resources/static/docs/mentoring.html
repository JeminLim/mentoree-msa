Host: localhost:8080

{
  "title" : "integrationTestProgram",
  "description" : "forTestProgram",
  "goal" : "integrationTest",
  "maxMember" : 3,
  "category" : "MUSIC",
  "dueDate" : "2022-09-25",
  "memberId" : 1,
  "memberNickname" : "testNick",

}

Content-Type: application/json
Content-Length: 94

{
  "participatedProgram" : {
    "id" : 36,
    "title" : "integrationTestProgram"

Request

Path Type Description

title

String

Program title to create

goal

String

Program goal

description

String

Program description

maxMember

Number

Maximum member can be able to participate program

category

String

Program’s category

dueDate

String

due date of recruitment

memberId

Number

Member pk who apply to create program

memberNickname

String

Member’s nickname

mentor

Boolean

Program role wants to participate

Response

Path Type Description

participatedProgram.id

Number

Created program id

participatedProgram.title

String

Created program title

programList[].dueDate

String

due date of recruitment

programList[].mentor[]

Array

List of mentor

hasNext

Boolean

Whether next page is exist or not
